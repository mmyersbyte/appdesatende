 Estrutura dos Arquivos
------------------------
5 arquivos principais de estilos, cada um dedicado a uma tela:

1. estilosLogin.js
   - Cuida da aparência da tela de login
   - Define estilos para botões de acesso
   - Configura o visual dos formulários de entrada

2. estilosHome.js
   - Estiliza a tela principal do app
   - Define layouts para listagem de empresas
   - Configura a barra de pesquisa e cards

3. estilosPerfil.js
   - Define o visual do perfil do usuário
   - Estiliza a área de reclamações
   - Configura modais e formulários de reclamação

4. estilosPerfilEmpresa.js
   - Cuida da aparência do perfil empresarial
   - Define estilos para listagem de reclamações
   - Configura respostas e interações

5. estilosConfiguracoes.js
   - Estiliza a tela de configurações
   - Define layouts para opções e ajustes
   - Configura elementos de personalização
 Padronização de Corez
- fundoPrincipal: Cor de fundo principal (background)
- corPrimaria: Cor principal de destaque
